Как мы можем найти элемент на странице

browser.find_element_by_tag_name - поиск элемента по названию тега browser.find_element_by_name - поиск элемента по содержимому в атрибуте name browser.find_element_by_class_name - поиск элемента по наименованию класса browser.find_element_by_id - поиск элемента по уникальному ключу browser.find_element_by_css_selector - поиск элемента по селекторам CSS browser.find_element_by_link_text - поиск ссылки по тексту (не рекомендуется) find_element_by_partial_link_text - поиск ссылки по части текста (не рекомендуется)
