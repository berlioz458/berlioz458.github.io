Как мы можем найти элемент на странице

Поиск по первому вхождению

Вернется первый найденый элемент

browser.find_element_by_tag_name - поиск элемента по названию тега
browser.find_element_by_name - поиск элемента по содержимому в атрибуте name
browser.find_element_by_class_name - поиск элемента по наименованию класса
browser.find_element_by_id - поиск элемента по уникальному ключу
browser.find_element_by_css_selector - поиск элемента по селекторам CSS
browser.find_element_by_link_text - поиск ссылки по тексту (не рекомендуется)
browser.find_element_by_partial_link_text - поиск ссылки по части текста (не рекомендуется)

Поиск всех необходимых

Вернется полный список элементов. Набор методов не отличается, разница лишь в отсутсвии find_element_by_id (логично).

browser.find_elements_by_css_selector; browser.find_elements_by_xpath; browser.find_elements_by_name; browser.find_elements_by_tag_name; browser.find_elements_by_class_name; browser.find_elements_by_link_text; browser.find_elements_by_partial_link_text.
